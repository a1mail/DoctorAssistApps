📋 Шаг 1: Анамнез и факторы риска (РК РФ 2021, раздел 2.1)

Сбор анамнеза — критически важный этап. Заполните все поля максимально подробно. Система автоматически рассчитает менопаузальный статус и оценит показания для BRCA-тестирования.

Полное ФИО пациента/ки для идентификации
При указании даты рождения возраст рассчитывается автоматически
Можно скорректировать вручную при необходимости
Мужской РМЖ всегда требует оценки BRCA и агрессивной тактики
Критично для выбора интенсивности лечения!

🩺 Шаг 2: Физический осмотр (РК РФ 2021, раздел 2.2)

Тщательное документирование физикального осмотра критично для оценки стадии. Система автоматически определит клиническую категорию T (cT) на основе введённых данных.

При двустороннем — документируйте каждую сторону отдельно

📌 Локальные жалобы

🌡️ Общие (системные) жалобы

📐 Характеристики опухоли

Наибольший диаметр при пальпации
Фиксация → критерий T4

🎯 Точная локализация опухоли (критично для планирования!)

УОК — наиболее частая локализация
12 часов = верхняя точка (сосок вверх)
Критично для планирования органосохраняющей операции
Множественность влияет на выбор объёма операции

🩹 Состояние кожи (критерии T4!)

Выбор T4-признаков → автоматически T4
Отёк >⅓ молочной железы = T4b
T4a = фиксация к грудной стенке

⚠️ ВЫЯВЛЕНЫ КРИТЕРИИ T4 (местнораспространённый рак)!

T4 подразделяется:
• T4a — фиксация к грудной стенке (рёбра, межрёберные мышцы)
• T4b — изъязвление, сателлиты, отёк >⅓
• T4c — T4a + T4b
• T4d — воспалительный рак (клинический диагноз!)

🎀 Состояние соска и ареолы

Недавняя ретракция — признак злокачественности
Требует биопсии соска при подозрении

🔬 Регионарные лимфатические узлы (критично для стадирования!)

Подмышечные (аксиллярные) лимфоузлы

Уровень влияет на cN-категорию
>2.5 см → N2, >5 см → вероятно N3
≥4 → вероятно N2-N3
Фиксация → признак N2-N3

Над- и подключичные лимфоузлы

Надключичные ЛУ → автоматически cN3c
Подключичные ЛУ → автоматически cN3a

Внутригрудные (парастернальные) лимфоузлы

Не пальпируются! Оценка только по визуализации

Заполните данные о лимфоузлах

🎯 Автоматическая оценка cT (клиническая)

Заполните размер опухоли
Классификация TNM-8:
Tis (in situ) • T1mi (≤0.1см) • T1a (0.1-0.5см) • T1b (0.5-1см) • T1c (1-2см) • T2 (2-5см) • T3 (>5см) • T4 (см. критерии выше)

🔬 Шаг 3: Диагностика (РК РФ 2021, раздел 2.3-2.5)

Комплексная диагностика включает визуализацию, морфологическую верификацию и молекулярное типирование. Все пункты обязательны для формирования диагноза и выбора лечения.

📸 Визуализация молочных желёз

Маммография (обязательна всем >40 лет)

BI-RADS ≥4 → показана биопсия
Может отличаться от клинического размера

УЗИ молочных желёз и регионарных зон

УЗИ — «золотой стандарт» измерения размера

МРТ молочных желёз (по показаниям)

Показания: молодой возраст, плотная грудь, BRCA+, неоадъювант
МРТ может переоценивать размер на 20-30%

🌐 Стадирование (исключение отдалённых метастазов)

Показания для стадирования (РК РФ 2021):
• Стадия IIA и выше (T2N0 или любая T с N+)
• Клинические/лабораторные признаки метастазов
• Агрессивные подтипы (TNBC, HER2+, Grade 3)

Показания: HER2+, TNBC, неврологические симптомы
ПЭТ-КТ — опция при сомнительных находках
SUV >2.5 — высоковероятно злокачественное

🤖 Автоматическая оценка cM (клиническая)

Заполните данные стадирования

🔬 Морфологическая верификация (обязательна!)

Core-биопсия (трепан-биопсия) — стандарт диагностики
NST = No Special Type (ранее «инвазивная протоковая»)
G3 → агрессивное течение, высокий Ki-67
Размер только инвазивной части (без DCIS)
Мультифокальность → ограничивает органосохранение
LVI → неблагоприятный прогностический фактор
EIC (extensive intraductal component) → выше риск рецидива

🧬 Иммуногистохимия (обязательная панель + расширенная)

Обязательные маркеры (ASCO/CAP 2020, РК РФ 2021):
• ER (рецепторы эстрогена) — % позитивных клеток
• PR (рецепторы прогестерона) — % позитивных клеток
• HER2 (ERBB2) — IHC 0/1+/2+/3+, при 2+ → FISH
• Ki-67 (индекс пролиферации) — %

ER+ ≥1% = чувствителен к гормонотерапии
Количественная оценка (0-100%)
HER2 3+ или FISH+ → таргетная терапия
Ki-67 <20% → Luminal A, ≥20% → Luminal B

🔬 Расширенная ИГХ-панель (по показаниям)

Мутация p53 → неблагоприятный прогноз
EGFR+ часто при TNBC
AR+ при TNBC → возможна антиандрогенная терапия
PD-L1 ≥1% при mTNBC → основание для добавления иммунотерапии

      Показания для BRCA-тестирования (NCCN 2024):
      • Возраст <45 лет • TNBC <60 лет • Семейный анамнез РМЖ/РЯ
      • Двусторонний РМЖ • Мужчина с РМЖ • Еврейское происхождение (Ashkenazi)

      Герминальные мутации BRCA1/2

      Кровь или слюна для анализа герминальных мутаций
      BRCA1+ → высокий риск РМЖ/РЯ
      BRCA2+ → высокий риск РМЖ (муж + жен)
      Для генетического консультирования семьи

      Панельное секвенирование (NGS) — расширенная панель

      Тестирование TP53, PALB2, CHEK2, ATM, CDH1 и др.
      PIK3CA+ при ER+/HER2- mBC → основание для алпелисиба
