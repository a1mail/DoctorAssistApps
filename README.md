# DoctorAssistApps

Applications which assist doctors in patient diagnosis, treatment and management.

## Oncology CDSS knowledge packs

- `cdss/knowledge/lung_cancer/` — Russian clinical guideline level lung cancer knowledge pack with YAML/DSL entities, inference, diagnosis, treatment, constraints, backward planning, scenarios, follow-up, workflow, and source traceability.
- `cdss/core/` — MVP loader and structural validator for YAML/DSL Clinical Knowledge Packs.
